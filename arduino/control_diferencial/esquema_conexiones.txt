/*
 * ESQUEMA DE CONEXIONES - Control Diferencial
 * ==========================================
 * 
 * Arduino Uno/Nano
 * ================
 * 
 * JOYSTICKS DIGITALES (3 estados):
 * ---------------------------------
 * Joystick Izquierdo:
 *   - Señal UP      → Pin 9
 *   - Señal DOWN    → Pin 10
 *   - Señal NEUTRAL → Pin 11
 *   - VCC           → 5V
 *   - GND           → GND
 * 
 * Joystick Derecho:
 *   - Señal UP      → Pin 12
 *   - Señal DOWN    → Pin A2
 *   - Señal NEUTRAL → Pin A3
 *   - VCC           → 5V
 *   - GND           → GND
 * 
 * DRIVER MOTORES (ej: L298N):
 * ---------------------------
 * Motor Izquierdo:
 *   - ENA (Enable)  → Pin 5 (PWM)
 *   - IN1           → Pin 4
 *   - IN2           → Pin 3
 *   - OUT1          → Motor Izq (+)
 *   - OUT2          → Motor Izq (-)
 * 
 * Motor Derecho:
 *   - ENB (Enable)  → Pin 6 (PWM)
 *   - IN3           → Pin 7
 *   - IN4           → Pin 8
 *   - OUT3          → Motor Der (+)
 *   - OUT4          → Motor Der (-)
 * 
 * ALIMENTACIÓN:
 * -------------
 * Arduino:
 *   - VCC → 5V (USB o fuente externa)
 *   - GND → GND común
 * 
 * Driver L298N:
 *   - VCC (Logic) → 5V Arduino
 *   - VCC (Motor) → 6-12V (fuente externa)
 *   - GND         → GND común
 * 
 * NOTAS IMPORTANTES:
 * ------------------
 * 1. Usar fuente externa para motores (6-12V)
 * 2. Conectar todos los GND juntos (común)
 * 3. Los joysticks digitales deben garantizar que solo una señal esté activa a la vez
 * 4. Configurar resistencias pull-down si los joysticks no las incluyen
 * 5. Verificar polaridad de motores para dirección correcta
 * 6. Si el robot gira al revés, intercambiar IN1-IN2 o IN3-IN4
 * 
 * DIAGRAMA ASCII:
 * ---------------
 * 
 *   [Joystick Izq]          [Joystick Der]
 *    UP  DOWN  NEU          UP  DOWN  NEU
 *     |    |    |            |    |    |
 *     9   10   11           12   A2   A3
 *     |    |    |            |    |    |
 *  +--+----+----+------------+----+----+--+
 *  |        ARDUINO UNO/NANO              |
 *  |                                      |
 *  |  Pin 3,4,5          Pin 6,7,8        |
 *  +----+---+----------------+---+--------+
 *       |   |                |   |
 *   +---+---+--+         +---+---+--+
 *   |  L298N   |         |  L298N   |
 *   | (Motor   |         | (Motor   |
 *   |  Izq)    |         |  Der)    |
 *   +----+-----+         +----+-----+
 *        |                    |
 *   [Motor Izq]          [Motor Der]
 * 
 */
