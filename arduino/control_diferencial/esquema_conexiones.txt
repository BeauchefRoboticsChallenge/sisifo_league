/*
 * ESQUEMA DE CONEXIONES - Control Diferencial
 * ==========================================
 * 
 * Arduino Uno/Nano
 * ================
 * 
 * JOYSTICKS DIGITALES (3 estados con 2 switches):
 * ------------------------------------------------
 * Joystick Izquierdo:
 *   - Switch UP     → Pin 9
 *   - Switch DOWN   → Pin 10
 *   - VCC           → 5V
 *   - GND           → GND
 * 
 * Joystick Derecho:
 *   - Switch UP     → Pin 11
 *   - Switch DOWN   → Pin 12
 *   - VCC           → 5V
 *   - GND           → GND
 * 
 * DRIVER MOTORES (ej: L298N):
 * ---------------------------
 * Motor Izquierdo:
 *   - ENA (Enable)  → Pin 5 (PWM)
 *   - IN1           → Pin 4
 *   - IN2           → Pin 3
 *   - OUT1          → Motor Izq (+)
 *   - OUT2          → Motor Izq (-)
 * 
 * Motor Derecho:
 *   - ENB (Enable)  → Pin 6 (PWM)
 *   - IN3           → Pin 7
 *   - IN4           → Pin 8
 *   - OUT3          → Motor Der (+)
 *   - OUT4          → Motor Der (-)
 * 
 * ALIMENTACIÓN:
 * -------------
 * Arduino:
 *   - VCC → 5V (USB o fuente externa)
 *   - GND → GND común
 * 
 * Driver L298N:
 *   - VCC (Logic) → 5V Arduino
 *   - VCC (Motor) → 6-12V (fuente externa)
 *   - GND         → GND común
 * 
 * NOTAS IMPORTANTES:
 * ------------------
 * 1. Usar fuente externa para motores (6-12V)
 * 2. Conectar todos los GND juntos (común)
 * 3. Cada joystick tiene 2 switches: UP y DOWN
 * 4. Los switches deben conectar el pin a GND cuando están activos (lógica invertida)
 * 5. Cuando ningún switch está activo, el estado es NEUTRAL
 * 6. Se utilizan resistencias pullup internas del Arduino (no se requieren externas)
 * 7. Verificar polaridad de motores para dirección correcta
 * 8. Si el robot gira al revés, intercambiar IN1-IN2 o IN3-IN4
 * 
 * DIAGRAMA ASCII:
 * ---------------
 * 
 *   [Joystick Izq]          [Joystick Der]
 *     UP   DOWN               UP   DOWN
 *      |     |                 |     |
 *      9    10                12    A2
 *      |     |                 |     |
 *  +---+-----+-----------------+-----+---+
 *  |        ARDUINO UNO/NANO               |
 *  |                                       |
 *  |  Pin 3,4,5          Pin 6,7,8         |
 *  +----+---+----------------+---+---------+
 *       |   |                |   |
 *   +---+---+--+         +---+---+--+
 *   |  L298N   |         |  L298N   |
 *   | (Motor   |         | (Motor   |
 *   |  Izq)    |         |  Der)    |
 *   +----+-----+         +----+-----+
 *        |                    |
 *   [Motor Izq]          [Motor Der]
 * 
 * ESTADOS DEL JOYSTICK:
 * ---------------------
 * - UP activo (LOW), DOWN inactivo (HIGH) = Estado UP
 * - DOWN activo (LOW), UP inactivo (HIGH) = Estado DOWN
 * - Ambos inactivos (HIGH, HIGH) = Estado NEUTRAL
 * 
 */
